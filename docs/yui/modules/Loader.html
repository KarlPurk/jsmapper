Loader Module

Defined in: packages\jsmapper\loader.js:25

This module basically allows me to keep the require paths relative to the jsmapper directory. We want to do this so that we can automatically convert those into variables when we build the project for the browser.

E.g: - require('./../../loader').load('/path/to/file')

instead of: - require('./file')

This allows us to convert: - require('./../../loader').load('/path/to/file')

into: - JsMapper.Path.To.File

which makes it easy to automatically build the project for use in the browser.
